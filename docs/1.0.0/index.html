Descrição

O componente gumga-number-in-words-ng é um filtro que retorna números em string, ou seja, escreve números por extenso.

Instalação

// npm
npm i gumga-number-in-words-ng --save

// bower
bower install gumga-number-in-words-ng --save
Usando

Ex 1: {{ '15.000' | gumgaNumberInWords:true }}

Ex 2: {{ ctrl.outroValor }}

  • HTML
  • JavaScript
<h1>Ex 1: {{ '15.000' | gumgaNumberInWords:true }}
<h1>Ex 2: {{ ctrl.outroValor }}</h1>
angular.module('app', ['gumga.numberinwords'])
  .controller('Ctrl', function($filter) {
    ctrl = this
    ctrl.outroValor = $filter('gumgaNumberInWords')('20', true)
  })
